# Junior Webmaster Camp 8
Web Register (Backend and Frontend) for Junior Webmaster Camp 8

## Installation
- nginx
- node.js & npm
- Docker
- MongoDB
- Node.js Package
- Set Enviroment

### Docker
[https://docs.docker.com/linux/step_one](https://docs.docker.com/linux/step_one)

```bash
npm install
```
